# Login Server

This is a a login server and a login client widget. 

Data Handling:
* HashMaps
* Java networking
* Basic encryption

GUI:
* JavaFX (client)
* Java Swing (server) 

## Installation

Clone this project. 

```bash
git clone https://github.com/Madelovina/Login-Server
```
